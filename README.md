# 🧬 Asthma Prediction using Machine Learning

## 📌 Overview

This project focuses on predicting asthma using clinical and demographic data through Machine Learning models. The system is designed to provide accurate and interpretable predictions for healthcare applications.

---

## ⚙️ Models Used

* Random Forest (Best Performing)
* XGBoost
* Neural Network

---
